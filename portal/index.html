Companion tools

Some Mazy tools live outside the browser — find them in the repository: cover_ed, a desktop app to edit audio metadata and artwork; srt_generator, a CLI that drafts SRT lyric timings from a song; and music_cache_updater, which keeps the player's metadata cache fresh.
