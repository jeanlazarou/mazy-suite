# Audio Mastering Tool

## Project Overview

Create a professional audio mastering application written in Go for audio files. The tool offers two interfaces:

1. **Command-Line Interface (CLI)**: For batch processing, automation, and power users
2. **Web Interface**: Modern React-based UI with Material UI for visual, interactive mastering, using the Go engine compiled in WebAssembly

The application should be production-ready, efficient, and user-friendly with a comprehensive preset system accessible from both interfaces.

It should also be able to analyze the audio file and suggest best matering, maybe depending on the target listening expirience (headphones, car audio system, etc.).

The Web Interface should help users listen to different mastering. It could display different visual components to help users understand the impact and interact graphically to tune the mastering, both using traditional physical devices but also intuitive panel where users can interact using their mouse.
